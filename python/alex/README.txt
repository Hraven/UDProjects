alex in een vitual assistent die werkt met spraak herkening en keyboard input


ze begint in typeing mode waar ze luistered naar keyboard input als je "switch" en "listen" in een zin om naar listen mode te schakelen,
zeg "switch" en "text" om terug naar text mode te gaan.

zeg alex in het begin van elke opdracht anders negeert ze het comando. elk comando word geloged in de folder interaction_logs, de log bevaten de tijd van het comando
en wat het comando was

ze werk met key words dus zolang de zin een keyword heeft zal ze reageren
